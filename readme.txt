A really simple wordpress plugin to grant users "redirection" permissions without giving other permissions

== Description== 

This plugins needs the [Redirection Plugin](https://redirection.me) and the [Members Plugin](https://wordpress.org/plugins/members/)
The purpose of this plugin  is to allow user to use the [Redirection Plugin](https://redirection.me)by adding a new capability with the help of the [Members Plugin](https://wordpress.org/plugins/members/).
